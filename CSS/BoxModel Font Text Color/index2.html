Learn about colours

colors can be represented as:

  1. color keywords
    2. //color: brown
  2. Hex color code
    3. //color: #543612
  3. RGB
    4. //color:rgb(23,45,67)
  4. RGBA
    5. //color:rgba(23,45,67,0) "0 to 1 is the alpha value which means opacity"
  5. HSL
    6. //color:hsl(255,100%,50%)
